# Student Expense & Budget Management System

This is a Python-based console application developed to help students manage their daily expenses and monthly budget efficiently.

## Features
- Add daily expenses
- View all expenses
- Set and update monthly budget
- View monthly spending summary
- View category-wise summary
- Delete expense records

## Technologies Used
- Python
- SQLite (Database)

## How to Run
1. Install Python
2. Download the file
3. Open terminal in the folder
4. Run:
   python expense_manager.py
